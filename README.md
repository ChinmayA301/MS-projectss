# MS And Later Code Projects

Curated MS-era and later code/project material recovered from Google Drive.

This repo is intentionally code-focused. Academic reports, third-party tutorial clones, vendored dependencies, local virtual environments, and generated build outputs were excluded unless they directly support an organic project.

## Projects

- `capstone/` - Capstone notebooks, environment file, instructions, and selected model/analysis outputs.
- `ai-sequence-project/` - Spring 2025 AI sequence project scripts for Bayesian optimization, reinforcement learning, trend propagation, graph modeling, and Streamlit UI work.
- `ml-programming/` - ML programming coursework implementations including perceptron, k-means/PCA, and MLP work.
- `rag/` - RAG banking script.
- `geo-visualizer/` - React/TypeScript geospatial visualizer source.
- `severance-style-visualizer/` - React/TypeScript visualizer source.

## Excluded

- `aegis-backend/` per request.
- Third-party tutorial/downloaded code such as the generic chatbot deployment tutorial clone.
- `node_modules/`, virtual environments, `.git/`, build outputs, installers, and local secret files.
