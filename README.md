# MS and Later Project Catalog

Curated graduate-era and later project work across data science, ML, AI systems, document intelligence, geospatial/service analytics, and applied data pipelines.

This repository is a catalog of coursework and self-directed builds. The strongest standalone portfolio projects have their own repositories; this repo preserves the surrounding technical evidence and smaller applied workflows.

## Portfolio Signals

- **Data science:** clustering, segmentation, feature engineering, model evaluation, and notebook-to-script packaging.
- **AI engineering:** RAG prototypes, document-intake benchmarks, and sequence/agent-style experimentation.
- **Analytics engineering:** public-data pipelines, reproducible scripts, and structured outputs.
- **Applied problem framing:** civic services, commercial real estate, social analytics, and banking-style retrieval.

## How To Read This Repo

Use this as supporting evidence rather than a single product. Recruiter-facing highlights:

- For **Data Scientist** roles: `civic-service-segmentation/`, `ml-programming/`, `social-media-growth-analytics/`, and `capstone/`.
- For **AI Engineer** roles: `document-intake-public-benchmarks/`, `rag/`, and `ai-sequence-project/`.
- For **Analytics / Data Engineering** roles: `commercial-real-estate-data-pipeline/` and any project with reusable scripts or structured outputs.
